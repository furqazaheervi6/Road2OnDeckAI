# Road2OnDeckAI

Road2OnDeckAI is the workspace for turning agentic computer-vision ambition into weekly shipped evidence.

## Mission

Become useful on agentic computer-vision work while protecting grades, health, and the biophysics path.

The point is not to look busy. The point is to become the kind of person who can understand a messy visual problem, build a working model or tool around it, communicate the tradeoffs clearly, and keep enough stability in school and life to stay in the game.

## Workspace Map

- `foundations/` - Python, PyTorch, math, systems, and computer-vision fundamentals.
- `projects/` - end-to-end builds, experiments, demos, and shipped artifacts.
- `notes/` - mission control, study notes, decisions, retrospectives, and planning.
- `portfolio-assets/` - screenshots, videos, figures, writeups, diagrams, and publishable proof.

## OnDeck-Ready Bar

This repo is OnDeck-ready when it shows credible evidence of:

- Python fluency: clean scripts, notebooks, CLIs, data handling, tests, and debugging habits.
- PyTorch training loops: datasets, dataloaders, models, losses, checkpoints, metrics, and reproducible runs.
- CV/video competence: image/video loading, transforms, labeling, inference, tracking or temporal reasoning, and practical evaluation.
- End-to-end ownership: scoped problem, working prototype, iteration log, results, failure analysis, and deployment or demo path.
- Polished communication: concise README, visuals, demo video, technical writeup, and clear next steps.

## Weekly Operating Rhythm

1. Choose one foundation block, one project block, and one grade or health guardrail.
2. Ship visible evidence before adding scope.
3. Capture decisions in `notes/index.md`.
4. Run the weekly retrospective from `notes/templates/weekly-retrospective.md`.
5. Move the best proof into `portfolio-assets/`.
