# Foundations

This lane is for the skills that make agentic computer-vision work possible.

## Focus Areas

- Python fluency and software hygiene
- NumPy, data loading, plotting, and debugging
- PyTorch tensors, autograd, modules, training loops, checkpoints, and metrics
- Computer vision fundamentals: images, video, transforms, augmentation, detection, segmentation, tracking, and evaluation
- Practical ML engineering: reproducibility, experiment notes, and failure analysis

## Evidence To Keep Here

- Small exercises with explanations
- Reusable snippets
- Training-loop walkthroughs
- CV/video mini-labs
- Notes that turn into project decisions
